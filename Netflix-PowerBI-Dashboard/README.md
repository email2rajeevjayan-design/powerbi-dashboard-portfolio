# Netflix Content Analytics Dashboard

## Project Overview

This dashboard analyzes Netflix content data to identify trends across genres, countries, ratings, and release years.

## Tools Used

- Power BI
- SQL
- Excel

## Key KPIs

- Total Titles
- Movies
- TV Shows
- Countries
- Genres
