# Healthcare Operations Analytics Dashboard

## Project Overview

This dashboard analyzes healthcare operational data to monitor patient admissions, department performance, and patient flow.

## Tools Used

- Power BI
- SQL
- Excel

## Key KPIs

- Total Patients
- Admissions
- Department Performance
- Average Length of Stay
